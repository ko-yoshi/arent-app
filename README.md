用意できたページは下記となります。

```bash

/column
```

[デプロイ先]()

## 開発-初回

```bash
git clone https://github.com/ko-yoshi/arent-app.git
cd arent-app
npm install
```

### 起動

```bash
npm run dev
```

### lint

```bash
npm run lintfix
```
